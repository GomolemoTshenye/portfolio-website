## 🚀 Getting Started

### View Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/GomolemoTshenye/portfolio.git
   cd portfolio
   ```

2. **Open in your browser**
   ```bash
   # Simply open index.html in your browser, or use Live Server in VS Code
   open index.html
   ```

3. **Using VS Code Live Server** *(recommended)*
   - Install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
   - Right-click `index.html` → **Open with Live Server**
   - Runs on port `5501` by default (see `settings.json`)

### Deploy to GitHub Pages

1. Push your code to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`

## 📂 Pages Overview

### 🏠 Home (`index.html`)
Hero section with a gradient background, profile photo, and a call-to-action button linking to the projects page.

### 👤 About (`about.html`)
Personal bio highlighting my academic background at **Varsity College (IIE)**, technical skills, and soft skills. Includes a skills grid and a CV download button.

### 💼 Projects (`projects.html`)
Showcases three key projects with images, descriptions, and technology stacks:
- **Cybersecurity Awareness Chatbot** — Python, JavaScript, HTML, CSS
- **Mobile-Friendly Website** — HTML, CSS, JavaScript
- **Azure-Integrated Web App** — C#, ASP.NET Core MVC, Azure

### 📬 Contact (`contact.html`)
Contact form with name, email, and message fields, plus direct links to email, GitHub, and LinkedIn.
